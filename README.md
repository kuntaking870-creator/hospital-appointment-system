# Hospital Appointment Attendance & Analytics System

## Overview

A production-ready enterprise-grade web application designed to monitor appointment attendance, provide real-time operational dashboards, and generate comprehensive statistics for hospital management.

## Key Features

- Automated HMS Integration
- Real-time Dashboard
- Advanced Search Module
- Comprehensive Reporting
- Role-Based Access Control

## Technology Stack

- Backend: ASP.NET Core 8
- Frontend: React + TypeScript
- Database: SQL Server
- Infrastructure: Docker

**Version**: 1.0.0
